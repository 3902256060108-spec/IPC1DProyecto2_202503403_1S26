El programa GameZone Pro es una aplicación desarrollada en Java que simula el funcionamiento de una plataforma integral de gestión para una tienda de videojuegos. A través de una interfaz gráfica construida con Java Swing, el usuario puede navegar entre diferentes módulos que representan las principales funcionalidades del sistema. En el módulo de tienda, se permite visualizar un catálogo de juegos, aplicar filtros por género y plataforma, realizar búsquedas por nombre o código, y agregar productos a un carrito para posteriormente confirmar la compra. Cada compra realizada se registra en un historial y también contribuye al sistema de recompensas del usuario.

El sistema de recompensas está basado en la acumulación de experiencia (XP), la cual aumenta conforme el usuario realiza acciones como comprar juegos, participar en torneos o coleccionar cartas. A medida que se acumula XP, el usuario sube de nivel y puede desbloquear distintos logros, los cuales se muestran visualmente en la interfaz. Estos logros permanecen guardados y forman parte del progreso del usuario dentro del sistema.

Además, el programa incluye un módulo de torneos en el cual los usuarios pueden inscribirse a diferentes eventos. Este módulo implementa el uso de estructuras de datos como colas para gestionar la atención de participantes, y utiliza hilos (threads) para simular el funcionamiento simultáneo de múltiples taquillas que venden tickets, permitiendo representar un sistema concurrente donde varios usuarios son atendidos al mismo tiempo.
